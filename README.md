# พิกัดหน่วยเลือกตั้ง (เว็บแอป)

เว็บแอปแบบไฟล์ล้วน (Static) เปิดได้ทั้งบนคอม/มือถือ  
- ค้นหา / กรอง หมู่-เขต-หน่วย
- ปักหมุดบนแผนที่ (Leaflet + OpenStreetMap)
- กด “นำทาง” เพื่อเปิด Google Maps

## วิธีเปิดใช้งาน (แนะนำ)
เพราะเบราว์เซอร์บางตัวไม่ยอมให้ fetch ไฟล์ JSON ตรง ๆ ถ้าเปิดแบบดับเบิลคลิก

### วิธีที่ 1: ใช้ Python (ง่ายสุด)
1) ติดตั้ง Python
2) เปิดโฟลเดอร์นี้ใน Terminal/Command Prompt
3) รัน:
   python -m http.server 8000
4) เปิดเว็บ:
   http://localhost:8000

### วิธีที่ 2: VS Code Live Server
ติดตั้ง Extension: Live Server → คลิกขวา index.html → Open with Live Server

## แก้ไขข้อมูล
- ข้อมูลอยู่ที่ `data.json`
- ถ้าต้องการอัปเดตจาก Excel: ทำใหม่ด้วยสคริปต์/หรือให้ฉันช่วยแปลงไฟล์เวอร์ชันใหม่ได้

> หมายเหตุ: แผนที่ต้องมีอินเทอร์เน็ตเพื่อโหลด tile แต่ตัวเว็บแอป/รายการ/ค้นหาถูก cache ได้
